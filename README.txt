This zip file contains an example project with a catapult that you can load.

How to get here from the previous .zip
1. Create a catapult blueprint based off the actor parent
2. Add catapult meshes and box collision trigger components
3. Make the trigger cover the entire catapult in the viewport window (if the blueprint page is emtpy, click on the catapult in the world and then click Edit Blueprint -> Apply Instance Changes to Blueprint -> reopen blueprint screen)
4. Add blueprint to character to make a button for loading
5. Add blueprint to catapult for loading
